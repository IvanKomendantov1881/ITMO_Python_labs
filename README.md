# ITMO Python Labs

Repository containing laboratory works completed during the Python programming course at ITMO University.

## About

This repository includes 10 laboratory works covering core Python programming concepts, algorithms, desktop and web development, computer vision, APIs, and speech technologies.

---

## Laboratory works

| Lab | Topic                       | Technologies               |
|-----|-----------------------------|----------------------------|
| 01 | Console Basics              | Python                     |
| 02 | Tabular Data Processing     | CSV, XML                   |
| 03 | Desktop GUI                 | Tkinter                    |
| 04 | Bags Problem                | Dynamic Programming        |
| 05 | Regular Expressions         | Regex                      |
| 06 | Object-Oriented Programming | OOP                        |
| 07 | REST API                    | Requests, JSON             |
| 08 | Computer Vision             | OpenCV                     |
| 09 | Flask Web Application       | Flask, HTML/CSS            |
| 10 | Voice Assistant             | Speech Recognition, Python |
